# `@doist/prettier-config`

> Doist global [prettier](https://prettier.io) config.

## Usage

**Install**:

```sh
npm install --save-dev @doist/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@doist/prettier-config"
}
```

## Release a new package

```
npm version [major | minor | patch]
npm publish
git push
```
